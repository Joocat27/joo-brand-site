# JOO — Personal Brand Site

도시를 읽고, 이야기를 만듭니다. (Cities, read & retold)

부동산·도시 연구자이자 크리에이터 JOO의 원페이지 개인 브랜드 사이트입니다.

## 구성

- `index.html` — 사이트 전체 (HTML + CSS + JS 단일 파일, 빌드 불필요)
- 3D 도시 모델: Three.js r128 (CDN 로드)
- 폰트: Pretendard, Noto Serif KR, IBM Plex Mono (CDN 로드)

## 로컬에서 보기

`index.html`을 브라우저로 열면 됩니다. (인터넷 연결 필요 — CDN 리소스)

## 배포 (GitHub → Vercel)

1. 이 폴더를 GitHub 저장소로 푸시
2. [vercel.com](https://vercel.com) → **Add New → Project** → 저장소 Import
3. Framework Preset: **Other**, Build 설정 없이 **Deploy**
4. 이후 `git push`할 때마다 자동 재배포
